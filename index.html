주요 기능

특허 검색 기능이외에도 다양한 보조 기능을 제공합니다!

검색식 생성

검색식만 사용해도 정확도는 높이면서 검색량을 확연하게 줄일 수 있습니다. CCPatents를 사용하면 몇 번의 클릭만으로 검색식을 쉽게 생성하고 변경할 수 있습니다.

동의어 사전

사용자가 추가한 키워드-동의어는 내부 데이터베이스에 자동으로 저장되고, 이후 사용자가 키워드를 검색하면 자동으로 동의어를 검색합니다. 기본적으로 키프리스 동의어사전과 워드넷을 내장하고 있지만, 노이즈가 많아서 추천드리지는 않습니다.

검색기록

사용자가 검색했던 내용을 그대로 저장하고 복구할 수 있습니다. 또한 검색 기록별로 중요한 출원번호를 메모하거나 평점을 매길 수 있습니다.

CCPatents

지원하는 검색 사이트

CCPatents는 자체 데이터베이스와 검색엔진이 없습니다. 대신 기존의 서비스를 더 쉽게 이용할 수 있게 해 줍니다.

Google Patents

구글에서 전 세계 80억 개의 특허 데이터베이스를 구매해서 여기에 구글 검색엔진을 적용했습니다! 직관적인 인터페이스는 덤! 특히 도면 보기가 정말 편합니다. 이미 미국 특허사무소의 70%는 구글 특허로 전환을 했다는 이야기가 있습니다. 단점은 각국 특허청이 제공하는 특허 검색 사이트에 비해 최신 DB가 즉시 반용되지 않는다는 점!

키프리스

한국 특허청에서 제공하는 공식 특허 검색 사이트입니다. 공기관에서 제공하는 것 치고는 준수한 인터페이스와 성능을 자랑합니다. 하지만 검색 결과를 PDF로 제공해서 보기가 불편하다는 단점이 있습니다. CCPatents는 키프리스 국내 특허/상표뿐만 아니라 해외 특허/상표 검색까지 모두 지원합니다.

USPTO

키프리스와 비슷하게 미국 특허청에서 제공하는 특허 검색 사이트입니다. 가장 최신의 미국 특허를 검색할 수 있다는 장점이 있습니다만, 매우 느리다는 단점이 있습니다. 특이하게도 등록된 특허 검색 사이트와 공고된 특허 검색 사이트가 따로 제공됩니다.

TESS

미국인들은 분리하는 것을 좋아하는 군요. 미국 상표 검색도 TESS라는 독립적인 서비스로 제공합니다. 여기는 뚫는데 시간이 좀 걸리긴 했는데, 어쨌거나 해냈습니다.

J-PlatPat

키프리스와 비슷하게 일본 특허청에서 제공하는 특허 검색 사이트입니다. CCPatents는 J-PlatPat만 검색결과를 즉시 표시하지 않습니다. 최초 1회만 J-PlatPat 사이트로 이동하고, 특허 검색 버튼을 누르면 검색식만 복사됩니다. 복사된 검색식은 직접 붙여넣기해야 합니다. 다른 사이트는 검색 결과 표시까지 자동으로 됩니다.

Espacenet

키프리스와 비슷하게 유럽 특허청에서 제공하는 특허 검색 사이트입니다.

CCPatents 사용 효과

특허 명세서에는 제목, 요약, 청구항이라는 중요한 필드가 있습니다. 특정 검색범위 마다 반드시 포함되어야 할 단어(+동의어)를 설정해 주는 것만으로도 정확도는 높이면서 검색량은 획기적으로 줄일 수 있습니다.
이러한 조건을 설정하기 위해 검색식이 사용되는데요.
CCPatents를 사용하면 클릭 몇 번으로 쉽게 검색식을 생성하고 변경할 수 있습니다.

검색식 생성1

추가된 단어는 정해진 위치로 알아서 이동!

그런데 왜 이렇게 단어가 많이 필요한가요?

원하는 특허를 제대로 검색하기 위해서는 한 두개의 키워드로는 어림도 없습니다. 예를 들어 '자석 필터'라고 입력하면, 검색 엔진은 자석 필터를 이용한 제품인지, 자석 필터를 만든는 방법인지, 특수한 자석 필터인지 알 방법이 없습니다. 그냥 수 만 건의 검색 결과만 뱉어낼 뿐입니다.

그리고 특허 명세서에는 일상 생활에서는 잘 사용되지 않는 단어도 많이 사용되기 때문에, 동의어를 반드시 입력해 주어야 합니다. 아래 특허 명세서는 폴더블 디스플레이에 관한 특허입니다. 그런데 정작 특허 제목은 '접이식 표시 장치'입니다.

검색식 생성2

클릭 몇 번으로 검색범위를 변경해 보자!

동의어 입력창 별로 다수의 검색 범위(제목, 요약, 청구항 또는 전문)를 선택할 수 있습니다.

CCPatents의 특허 일부입니다.
주요 단어가 제목, 요약, 청구항에 집중되어 있습니다.

동의어 사전

한 번 추가한 단어는 이후에 자동으로 검색!

CCPatents 내부에 키워드-동의어 쌍을 저장하는 데이터베이스가 있습니다.
그리고 여러분이 입력한 키워드-동의어 쌍은 현재 CCPatents에서 수집하고 있습니다. 딱 이것만 수집합니다.
이 데이터를 가공하여 나중에 더 좋은 기능을 추가하도록 하겠습니다.

검색 기록

검색 기록별로 메모와 평가를 할 수 있어요.

일단 요약 위주로 보다가 많이 비슷해 보이는 특허를 발견하면 출원번호 정도만 CCPatents에 메모해 두고, 나중에 한번에 모아서 분석해 보세요.
